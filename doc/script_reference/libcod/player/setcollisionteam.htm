setCollisionTeam( <team> )

Module: Player

Summary:

Set the team(s) which the player should have character collision with. No collision has precedence in case two players with different settings would collide with each other. Crosshair names (see cg_drawCrosshairNames client dvar) disappear for players that have a different setting than 'both'.



Call this on:

A Player



Example:

self setCollisionTeam("axis");



Required Args:

  • 1 : <team> A string description of the team. Valid teams are 'axis', 'allies', 'none' or 'both'
