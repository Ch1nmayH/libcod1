Remote Command

Module: Callbacks

Summary:

Calls CodeCallback_RemoteCommand() in maps/mp/gametypes/_callbacksetup.gsc when an authenticated remote (rcon) command is received. Requires self remoteCommand(from, messagePointer); to be executed for the command to be processed as usual. Commands not causing this callback to be invoked are:

  • map
  • devmap
  • map_restart
  • fast_restart

Note: The existance of the callback function is checked only once on server start. Functions added later are not taken into account, even after a map switch.



Parameters:

  • 1 : <from>: IP address of the sender.
  • 2 : <message>: Received command.
  • 3 : <messagePointer>: Pointer to the internal message structure (type msg_t).


Example:

CodeCallback_RemoteCommand(from, message, messagePointer)
{
    // Parse command for further actions
    remoteCommand(from, messagePointer);
}
