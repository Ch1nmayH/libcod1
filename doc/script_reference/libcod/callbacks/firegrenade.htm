Fire Grenade

Module: Callbacks

Summary:

Calls CodeCallback_FireGrenade() in maps/mp/gametypes/_callbacksetup.gsc after a grenade was thrown (right after grenade entity spawn).

Note: The existance of the callback function is checked only once on server start. Functions added later are not taken into account, even after a map switch.



Called on:

The throwing entity (the player or an <entity> from addTestClient())



Parameters:

  • 1 : <grenade>: The spawned grenade entity.
  • 2 : <name>: Weapon file name of the grenade.


Example:

CodeCallback_FireGrenade(grenade, name)
{
    iprintLn(self.name + "^7 threw a " + name);
}
