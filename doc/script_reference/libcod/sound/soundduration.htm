soundDuration( <aliasname> )

Module: Sound

Summary:

Returns the sound's duration in seconds (float), if the sound alias exists.

Note: This function can cause lags if used during gameplay. Therefore, it is recommended to call it right on map load to collect all required information.

Note: Similar to soundExists, this function returns undefined if the current map is not defined in the sound alias' loadspec.



Example:

duration = soundDuration("ctf_touchown");



Required Args:

  • 1 : <aliasname> The sound alias to get the duration of
