spawnGrenade( <weapon>, <origin>, [<direction>], [<velocity>], [<fuseTime>] )

Module: Weapons

Summary:

Spawns a new grenade and returns a reference to the grenade.



Call this on:

The attacking entity



Example:

//attacker = spawn("script_origin", (0, 0, 0)); // world as attacker
attacker = self;
origin = self getTagOrigin("tag_weapon_right");
dir = anglesToForward(self getPlayerAngles());
velocity = maps\mp\_utility::vectorScale(dir, 512);
fuseTime = 3;
attacker spawnGrenade("frag_grenade_german_mp", origin, dir, velocity, fuseTime);



Required Args:

  • 1 : <weapon> The name of the weapon item to spawn (constant string)
  • 2 : <origin> The position where the grenade is to be spawned (vector)


Optional Args

  • 1 : <direction> The orientation in which the grenade is to be spawned (vector)
  • 2 : <velocity> The grenade's velocity on spawn (vector)
  • 3 : <fuseTime> The remaining fuse time of the grenade in seconds (float)
