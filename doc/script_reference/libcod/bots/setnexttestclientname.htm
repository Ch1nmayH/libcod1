setNextTestClientName( <name> )

Module: Bots

Summary:

Defines the name of the next test client. Must be called before calling addTestClient() to take effect. Use resetTestClientNaming() to revert to the stock naming scheme.



Example:

setNextTestClientName("Unknown Soldier");
ent = addTestClient();



Required Args:

  • 1 : <name> The name the next test client will have.
