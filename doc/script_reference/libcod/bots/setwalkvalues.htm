setWalkValues( <forward_count>, <right_count> )

Module: Bots

Summary:

Order the bot to move in the specified direction.



Call this on:

<entity> from addTestClient()



Example:

bot setWalkValues(127, 127);



Required Args:

  • 1 : <forward_count> Forward direction of where the bot should move. Valid values: -128 - 127
  • 2 : <right_count> Right direction of where the bot should move. Valid values: -128 - 127
