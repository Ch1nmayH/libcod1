remoteCommand( <from>, <messagePointer> )

Module: System

Summary:

Processes a remote command that was passed to CodeCallback_RemoteCommand.



Parameters:

  • 1 : <from>: IP address of the sender.
  • 3 : <messagePointer>: Pointer to the internal message structure (type msg_t).


Example:

CodeCallback_RemoteCommand(from, args, messagePointer)
{
    remoteCommand(from, messagePointer);
}
