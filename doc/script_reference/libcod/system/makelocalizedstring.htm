makeLocalizedString( <string> )

Module: System

Summary:

Converts a regular string to a localized string (on the server side).



Example:

// If self is a player and level.hud_text a hud element:
text = "Welcome " + self.name;
level.hud_text setText(makeLocalizedString(text));



Required Args:

  • 1 : <string> The string to convert.
