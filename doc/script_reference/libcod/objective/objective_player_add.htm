objective_player_add( <objective_number>, <state>, <text>, <position>, <iconshader> )

Module: Objective

Summary:

Adds a new objective, visible for the player only. Overrides global objectives that use the same objective_number.



Call this on:

The player



Example:

self objective_player_add(objective_number, "current", origin);



Required Args:

  • 1 : <objective_number> The number of the objective to add
  • 2 : <state> A string value representing the state of the objective. Valid states are "empty", "invisible" and "current"


Optional Args

  • 1 : <position> The position of the objective
  • 2 : <iconshader> The objective icon to embed
