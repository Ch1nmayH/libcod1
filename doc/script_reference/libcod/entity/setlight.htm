setLight( <red>, <green>, <blue>, <intensity> )

Module: Entity

Summary:

Sets the dynamic light value for entities of type script_model. Visible in DirectX 9 mode only and with dynamic lights enabled (see the r_rendererInUse, r_rendererPreference and r_dlightLimit client dvars).



Example:

// If self is a player:
e = spawn("script_model", self.origin);
e linkTo(self);
e setLight(255, 0, 0, 255); // Red light



Required Args:

  • 1 : <red> Amount of red color (0-255)
  • 2 : <green> Amount of green color (0-255)
  • 3 : <green> Amount of blue color (0-255)
  • 4 : <intensity> Light intensity (0-255)
